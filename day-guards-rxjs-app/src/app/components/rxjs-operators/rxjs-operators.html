Advanced reactive programming patterns

map & filter

Transform and filter data streams

of(1,2,3,4,5).pipe( map(x => x * 2), filter(x => x > 5) ) // Output: 6, 8, 10

debounceTime

Wait for user to stop typing

Used in search component - waits 300ms after typing

searchTerms.pipe( debounceTime(300), distinctUntilChanged() )

switchMap

Cancel previous requests

Cancels old searches when new search starts

searchTerms.pipe( debounceTime(300), switchMap(term => this.search(term)) )

combineLatest

Combine multiple observables

Merges stock and sensor data

combineLatest([ stockData$, sensorData$ ]).pipe( map(([stocks, sensors]) => (...)) )

scan

Accumulate values over time

Like Array.reduce but for streams

events$.pipe( scan((acc, event) => [...acc, event], []) slice(-10) )

shareReplay

Share subscription & cache value

Multiple subscribers share one stream

data$.pipe( shareReplay(1) ) // Prevents multiple API calls
